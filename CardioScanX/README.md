CardioScanX
=============

This is a Streamlit app for mass screening of ECGs. It uses pre-trained models from the original repository and only performs inference.

How to run:

- Create a Python environment and install packages:

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

- Run Streamlit:

```bash
streamlit run app.py
```
